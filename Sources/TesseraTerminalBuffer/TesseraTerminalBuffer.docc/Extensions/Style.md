# ``Style``

The complete style associated with a cell: text colors, SGR text attributes, underline styling, and an optional OSC 8 hyperlink.

## Topics

### Creating a style

- ``init(foreground:background:attributes:underlineStyle:underlineColor:hyperlink:)``

### Setting the foreground

- ``foreground``

### Setting the background

- ``background``

### Applying text attributes

- ``attributes``
- ``TextAttributes``

### Configuring underlines

- ``underlineStyle``
- ``underlineColor``

### Associating a hyperlink

- ``hyperlink``
