# ``TesseraTerminal``

The public import surface for Tessera's terminal foundation.

## Overview

Import `TesseraTerminal` when building directly on the terminal substrate. The target
re-exports ANSI, buffer, input, I/O, and rendering modules.

## Topics

### Platform

- ``/TesseraTerminalIO/PlatformIO``

### Foundation modules

- ``/TesseraTerminalANSI``
- ``/TesseraTerminalBuffer``
- ``/TesseraTerminalCore``
- ``/TesseraTerminalInput``
- ``/TesseraTerminalRendering``
